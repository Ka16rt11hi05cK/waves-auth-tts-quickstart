bash

  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Accept: audio/wav" \
  -d '{
        "text": "Hello from Waves! This is your first Text to Speech API request.",
        "voice": "arjun",
        "language": "en"
      }' --output output.wav
If successful, the audio is saved as output.wav.

üì• Example (JavaScript)
javascript
Copy code
const fs = require("fs");
const fetch = require("node-fetch");

async function tts() {
  const response = await fetch("https://waves.smallest.ai/v4/tts", {
    method: "POST",
    headers: {
      "Authorization": "Bearer YOUR_API_KEY",
      "Content-Type": "application/json",
      "Accept": "audio/wav"
    },
    body: JSON.stringify({
      text: "Hello from Waves!",
      voice: "arjun",
      language: "en"
    })
  });

  const buffer = Buffer.from(await response.arrayBuffer());
  fs.writeFileSync("output.wav", buffer);
}

üîä Text-to-Speech API Reference {#tts-endpoint}

text	string	Yes	Input text to convert to speech

language	string	Yes	Language code (e.g., "en")
emotion	string	No	Optional emotion (e.g., "happy", "sad")
speed	number	No	Speaking rate (0.5‚Äì2.0, default=1.0)
format	string	No	"wav" or "mp3"
